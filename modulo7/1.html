O que é JavaScript?

O JavaScript foi criado para tornar as páginas web interativas executando scripts no navegador. No navegador, ele roda em um ambiente chamado sandbox que limita seu acesso ao sistema, já nom servidor ele tem acesso ilimitado ao sistema e varias funcionalidades como acesso ao banco de dados arquivos do sistema entre outros. Objetos globais como document, console e alert permitem interagir com a página, document ele aponta para o dom registrar mensagens e exibir alertas.
